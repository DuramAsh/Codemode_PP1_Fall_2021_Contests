Description
There is a country Darmengolia. This country is known for its bad army. 
Because the commander of this army prefers to put the weak ahead of the army, and the strong at the end.
So, unfortunately you were born in this country and became the coordinator of the army. 
And you need to position the army by "Darmenov" method.

You get N, which means 2D array with size N x N. Then you should fill this 2D array correctly.

Input : N - size of 2D array

Constraint : N > 0. If N == 0, print "Ne chetko"

Output : 2D array with size N x N
